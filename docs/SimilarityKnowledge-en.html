The Similarity Knowledge Ontology.
Ontology Specification Draft

Abstract

The similarity knowledge ontology defines four strategies that can be used when evaluating the similarity of two concepts defined in iSeeOnto. These following classes defined as subclasses of the Similarity Strategy class: - Exact Match, which indicates that there is a match if two individuals are exactly the same individual. - Hierarchy Distance, which indicates that two individuals match if they share a common type, but the similarity score depends on the distance between the asserted type of the type individuals. - Hierarchy Match, which indicates that two individuals match if they share a common type, regardless of its place in the hierarchy. - Same Type Match, which indicates that two individuals match if they both have the same asserted type.
